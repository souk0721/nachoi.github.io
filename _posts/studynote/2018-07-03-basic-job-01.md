# 필요 개념
  - `윈도우 작업 스케줄러` : 윈도우에서 사용자가 임의로 지정한 시간 및 방식에 따라서 자동으로 프로그램 및 스크립트를 사용하도록 해주는 윈도우 시스템 도구  

### When?
윈도우 시작 할 때나, 중복적인 시간 마다 프로그램을 실행 하고 싶을 떄 등록 해서 사용 합니다.
  
### How?
1. 작업 등록
 - `시작`->`실행`->`compmgmt.msc`엔터 
 - `시스템 도구`->`작업 스케줄러`에서 마우스 오른쪽 버튼 ->`작업 만들기`
